# 💰 AI-Powered Expense Tracker

An intelligent Expense Tracker Web Application built using **Python, Flask, SQLite, and Google Gemini AI**. The application allows users to manage daily expenses, visualize spending through dashboards, and receive AI-powered financial insights and recommendations.

---

## 🚀 Features

- ➕ Add Expenses
- ✏️ Edit Expenses
- ❌ Delete Expenses
- 🔍 Search Expenses
- 📊 Interactive Dashboard
- 📈 Monthly Expense Charts
- 🥧 Category-wise Pie Chart
- 📑 Expense Reports
- 📥 Export Expenses to CSV
- 🤖 AI Expense Analysis using Google Gemini
- 💬 AI Financial Chatbot
- 📱 Responsive User Interface

## 📥 Installation

Clone the repository

```bash
git clone https://github.com/ritika280/AI-Expense-Tracker.git
```

Move into the project folder

```bash
cd AI-Expense-Tracker
```

Install dependencies

```bash
pip install -r requirements.txt
```

Create a `.env` file

```env
GEMINI_API_KEY=YOUR_API_KEY
```

Run the application

```bash
python newapp.py
```

Open in your browser

```
http://127.0.0.1:5000
```
